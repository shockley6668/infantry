# infantry
 
## 分层介绍
本框架主要分为三层，分别是hal库层，bsp层，app层

bsp层主要是把你们之前学的hal库操作封装成简单的函数，

app层主要实现控制机器人逻辑的开发，比如云台控制，底盘控制，发射控制

component中是一些算法的实现函数，比如pid控制器，滤波器的实现，虚拟串口的实现，通讯协议等
